
# Product Manager API
Mini API para gestionar productos - Curso de Metodologías de Desarrollo UISEK

## Tecnologías
- Python
- FastAPI
- Uvicorn
- Git / GitHub

## Ejecución
1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. uvicorn app.main:app --reload
